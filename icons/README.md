# App Icons

This directory should contain the following icon files for the PWA:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

For a production app, you would need to create actual PNG icon files at these sizes with your app logo.
